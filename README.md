
# 💎 Diamond Price Prediction

A Machine Learning web application that predicts the price of a diamond based on its physical and quality features. Built with Flask, Scikit-learn, and deployed on Render.

## 🚀 Live Demo
🔗 https://diamondpriceprediction-cwh9.onrender.com/

---

## 📌 Features

- Predict diamond prices instantly
- User-friendly web interface
- Machine learning model integration
- Responsive design
- Real-time input validation
- Fast deployment on Render

---

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- Flask

### Machine Learning
- Scikit-learn
- Pandas
- NumPy

### Deployment
- Render

---

## 📊 Input Features

The model predicts price based on:

- Carat
- Cut
- Color
- Clarity
- Depth
- Table
- X (length)
- Y (width)
- Z (depth)
